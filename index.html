HTML

HTML (HyperText Markup Language) is the code used to structure a web page and its content. It is the language of the World Wide Web. It consists of a series of elements that tells the browser how to display the content.

CSS Styling

CSS (Cascading Style Sheets) is the code that styles web content written in HTML or other ‘markup language’ which can include the layout, colours, fonts. There is Incline CSS, Internal or Embedded CSS, and External CSS.

JavaScript

JavaScript is a scripting and programming language of the Web. It can update and change both HTML and CSS as well as calculate, manipulate and validate data. JavaScript makes responsive design easier. For example, you can create dynamic content, control multimedia, animate images, build complexed interactions into desktop and mobile applications.

SQL

SQL (Structured Query Language) is a domain specific language used in programming and design for managing data held in a relational database wherein each row reflects a data entity, and every column defines a specific information field. The SQL is used to create, store, update, and retrieve data.
